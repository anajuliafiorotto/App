Inicio

O Gerenciador de Tarefas é um aplicativo inovador utilizado para criar listas de afazeres a fim de se organizar e otimizar seu tempo.


Para criar uma tarefa você deve:

1 - Clicar em Tarefas

2 - Clicar em Criar nova tarefa

3 - Digitar a tarefa que você deseja criar

4 - Clicar em Adicionar


Para alterar o status de uma tarefa você deve:

1 - Clicar na tarefa que você deseja alterar o status


Para editar uma tarefa você deve:

1 - Deslizar a tarefa que você deseja editar para a esquerda

2 - Clicar em Editar

3 - Digitar o novo nome da tarefa

4 - Clicar em Alterar


Para excluir uma tarefa você deve:

1 - Deslizar a tarefa que você deseja excluir para a direita

2 - Clicar em Excluir

3 - Clicar em Confirmar
